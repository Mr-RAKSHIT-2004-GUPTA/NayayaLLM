## Core Features:

- Document Input: Allows users to upload PDF or TXT files or paste document text.
- Mode Selection: Provides buttons for 'Ask Question', 'Section-wise Summary', and 'Complete Summary' modes.
- Question Input: Displays a text input field for legal questions in 'Ask Question' mode.
- API Submission: Submits the document and question to the backend API using multipart/form-data.
- AI Question Answering: Utilizes AI to answer user questions about the uploaded legal document; this is an AI tool that uses the document context to produce its answers.
- Section Summary Generation: Generates summaries for each section of the document using AI.
- Complete Summary Generation: Generates a complete summary of the document using AI.
- Result Display: Displays the AI-generated results in a clean, user-friendly format.

## Style Guidelines:

- Primary color: Deep navy blue (#1A237E) to evoke trust and professionalism.
- Background color: Light gray (#F5F5F5) for a clean, uncluttered interface.
- Accent color: Muted gold (#A3875D) for highlights and CTAs, symbolizing legal tradition and sophistication.
- Body text: 'Inter', a sans-serif font, for clean and modern readability.
- Headline text: 'Alegreya', a serif font, for a traditional and professional look.
- Use simple, clear icons related to document types and functions.
- Employ a clean, organized layout with clear sections for input, mode selection, and results.
- Use subtle loading animations and transitions to provide feedback without being distracting.
